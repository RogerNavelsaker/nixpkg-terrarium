# Terrarium

Use this skill when you need structured specs, planning documents, or handoff artifacts tracked in git.

Preferred commands:

```bash
terrarium --help
terrarium list
terrarium status
```

Rules:
- Use terrarium for durable planning artifacts.
- Keep specs and plans explicit, short, and versioned.
- Prefer updating the artifact in terrarium rather than restating the plan in chat.
